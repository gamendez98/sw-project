Descripción navegación

  • Inicio: Esta es la página principal del sitio web donde se muestra la información general.
  • Autor: En esta página se puede encontrar información sobre el autor ingresado en el cuadro de busqueda.
  • Tópico: Esta página muestra los articulos relacionados a la uri del tópico ingresada.
  • Campo de Estudio: Esta página muestra los articulos relacionados a la uri del campo de estudio ingresada.
  • Sugerencia de publicaciones: Página dedicada a sugerir publicaciones relevantes al usuario de acuerda a una uri.
  • Detalles de publicación: Muestra detalles específicos de una publicación en particular.
  • Publicaciones relacionadas: Si se ingresa el nombre exacto y se seleccionada busqueda no exhaustiva se retornan los datos del paper ingresado, si se selecciona busqueda exhaustiva y se da el nombre del paper bien o mal, retorna los papers más relacionados a ese nombre.
  • BarPlot Tópicos: Presenta un gráfico de barras con la distribución de tópicos.
  • Conexión entre Tópicos: Visualización de la conexión entre diferentes tópicos mediante un gráfico.
  • Crear: Página para la creación de nuevas publicaciones.
